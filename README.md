\# Cortez Masters (Telegram Mini App)



Монорепозиторий проекта: фронтенд на React (Vite) и бэкенд на NestJS.



\## Стек

\- Frontend: React, Vite, TypeScript, Socket.IO client

\- Backend: NestJS, TypeORM, SQLite, Socket.IO, Telegram Bot API



\## Запуск локально



\### Frontend

```bash

cd client

cp env.example .env

npm install

npm run dev
